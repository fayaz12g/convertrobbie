# BNTX Extractor v0.6
Extracts textures from the Binary NX Texture ('BNTX' / .bntx file extension) format used in Switch games, and saves them as DDS.  

## Requirements:
* Python 3.4 or higher.
* cx_Freeze. (Optional)

## Supported formats:

* R8_G8_B8_A8_SRGB
